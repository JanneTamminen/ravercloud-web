# RaverCloud — Virtual DJ Residency

Marketing landing page for the RaverCloud Founding DJ Residency program (Cohort 01).

## Stack

- [Next.js](https://nextjs.org/) 15 (App Router)
- Midnight theme from the design handoff (`design_handoff_ravercloud/`)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Connected to Vercel for automatic deployments on push to `main`.

## Design handoff

Original HTML/CSS prototype and implementation brief live in `design_handoff_ravercloud/README.md`.
